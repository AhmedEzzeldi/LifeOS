New Features 12
New
Personalised Welcome Screen
A full-screen onboarding popup greets new users, asks for their name, and stores it. The app uses this name in all dashboard greetings and the sidebar footer. Returning users skip straight to their app.
New
Goal Fireworks Celebration
Dragging a goal's progress slider to 100% triggers a full-screen multi-burst confetti animation with a trophy banner. Particles use gravity physics and varied shapes. The celebration includes the goal's title and can be dismissed manually.
New
Dark / Light Mode Toggle
A sun/moon button in the top bar switches between light and dark themes. Automatically detects the system preference on first load. Every surface, badge, shadow, and semantic color adapts seamlessly. Preference is saved to localStorage.
New
Planner Event Alarms
Events can now have alarms toggled on when created. A background checker runs every 30 seconds and fires on the exact scheduled minute with a triple-beep audio tone via Web Audio API, an in-app popup notification, and a native browser push notification (when permitted).
New
Clear All Button on Every Page
Each section — Tasks, Goals, Planner, Habits, and Notes — now has a red "Clear All" button in its action bar. Each asks for confirmation before wiping. Clearing Habits also wipes the full check-log history since there's nothing to track without the habits themselves.
New
Expanded Quote Library (60+)
The daily motivational quote pool grew from 12 to over 60 entries, sourced from athletes, entrepreneurs, authors, and philosophers. Quotes cycle on a deterministic daily formula so each day reliably shows a different quote.
New
In-App Toast Notifications
A non-blocking toast bar now appears at the bottom-right to confirm actions — focus timer completions, alarm permissions granted, and session resets — without interrupting the user's flow.
New
Focus Timer Audio Feedback
When a Pomodoro session finishes, a short two-tone chime plays via Web Audio API to alert the user even if they're not looking at the screen. No external sound files required.
New
Priority Color-Coded Task Borders
Task list items now have a left-edge color indicator — red for high priority, amber for medium, green for low — so urgency is immediately visible without reading the badge.
New
Goal Category Icons
Each goal card now shows an emoji icon based on its category (💼 Career, 💪 Health, 💰 Finance, 📚 Learning, 🌱 Personal, ❤️ Relationships, 🎨 Creative). Goals with no category default to 🎯.
New
Notification Permission Banner in Planner
If browser notifications are not yet enabled, the Planner shows a contextual banner with a one-click "Enable Alarms" button. If notifications are blocked entirely, a different message explains the fallback to in-app popups.
New
Time-Aware Dashboard Greeting
The dashboard greeting changes based on the current hour — morning, afternoon, evening, and late-night variants — and addresses the user by their chosen name for a personalised experience.

Bug Fixes 6
Fixed
Habits Disappearing After Being Checked
Checking a habit caused the entire panel to re-render, which wiped habits from the visible list despite the data still being stored. The toggle now surgically updates only that row's checkbox state without a full re-render — habits are always visible.
Fixed
Habit Counter Not Resetting When Habit Is Deleted
Deleting a habit left its ID in every date entry of the check-log (hlog), so the done counter kept counting it even though the habit was gone. Deletion now scrubs the ID from the entire hlog before saving.
Fixed
Hardcoded Footer Showing Short Name
The sidebar footer was showing only "Ahmed" because a JavaScript line was overwriting the hardcoded HTML with whatever name was typed at the welcome screen. The footer is now fully hardcoded in HTML and no JS can overwrite it.
Fixed
Streak Counter Showing Wrong Value After Unchecking
The streak calculation walked backwards from today using the previous state of the log, not the updated one. The function now reads directly from the freshly mutated hlog copy, so streaks always reflect the current state.
Fixed
File Not Saving to Outputs Directory
A previous build was silently failing to write to /mnt/user-data/outputs, meaning users were downloading a stale version. The build process was corrected and validated with a Node.js syntax check before delivery.
Fixed
AlarmFired Flag Not Resetting on New Day
Once an alarm fired, its alarmFired flag was never reset, so recurring events on future dates would never alarm again. On app init, all events with alarmFired = true where date is in the past are now reset.

Improvements 8
Improved
Fully Responsive on All Screen Sizes
The sidebar now collapses off-screen on mobile with a hamburger menu and backdrop overlay. Metric grids, navigation cards, and note grids all collapse to 2-column or single-column layouts on smaller screens. The planner week grid has a minimum width with horizontal scroll on phones.
Improved
Dashboard Focused on Overview Only
The dashboard was previously mixing other sections' actions inline. It is now a pure overview with live stats, year progress, quote, and jump-to navigation cards only. Each page owns its own actions exclusively.
Improved
Senior-Level Task List Design
Tasks are sorted by priority then due date. Completed tasks drop to the bottom. Each row has a left-border priority indicator, smooth animated checkbox, hover-reveal delete button, and graceful text truncation with ellipsis for long notes.
Improved
Motivational Microcopy Throughout
Timer completion messages are now personality-rich ("You just leveled up. 💪"). Goal deletion prompts are firm. Empty states have motivational taglines. The year progress bar includes a days-remaining motivational line.
Improved
Goal Progress Bar Updates Live
Dragging the progress slider no longer requires a full panel re-render. The bar width and percentage label update in real time as the slider moves, then fireworks trigger asynchronously only when 100% is reached.
Improved
Habits 21-Day History Dot Updates Instantly
The habit history dots for today's date now toggle on/off in sync with the checkbox via DOM attribute selectors, with no re-render needed. The "today" dot also has an orange ring outline to distinguish it from past days.
Improved
Planner Events Sorted by Time
Events in each day column are now sorted chronologically by their time field so the day's schedule reads top-to-bottom in order. Events with alarms show an ⏰ indicator in the pill.
Improved
Nav Badges Show Real-Time Counts
The sidebar Tasks badge shows the live count of active (non-done) tasks. The Habits badge shows today's check ratio (e.g. 2/5). Both update immediately after any action — no page navigation required.
🗑
Removed 3
Removed
Cross-Page Actions on Dashboard
The old dashboard contained "New Task", "My Goals", and "Planner" buttons that duplicated actions from other pages. These have been removed in favour of clean navigation cards that jump to each section.
Removed
Dynamic Sidebar Author Name
The author credit was previously being overwritten at runtime by the user's welcome-screen name input. The dynamic update has been removed — the credit is now a static HTML element that never changes.
Removed
Alert() Dialogs for Timer Completion
Blocking browser alert() dialogs were replaced with non-blocking toast notifications and audio feedback so the timer page re-renders cleanly without the user needing to click away an interrupting modal.
